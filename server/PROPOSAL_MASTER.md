## Rare Earth Brand Design System

Brand colors are derived from the actual company logo (`Logo-nobackground.png.png`), NOT the Ion PowerPoint template.

### Color Palette

| Token | Hex | Usage |
|---|---|---|
| Crimson Red | #CC1111 | Primary accent - heading borders, accent bars, cover accents |
| Lime Green | #6ABF1E | Secondary accent - cert badges, callout borders, success |
| Dark Cinematic | #111111 | Cover page background |
| Cream | #FAF8F5 | Body page background (warm off-white) |
| Near-black | #1A1A1A | Body text, table headers |
| Warm Gray | #6B6B6B | Secondary text, captions, metadata |
| Light Gray | #E8E6E3 | Table alt rows, subtle borders |
| White | #FFFFFF | Cover text, card backgrounds |
| Gold | #D4A84B | Sparingly - highlights, star accents |

### Typography

- Headings: "Bookman Old Style", "Georgia", "Times New Roman", serif
- Body: "Bookman Old Style", "Georgia", "Times New Roman", serif
- Sans-serif accent (cert badges, eyebrow labels, footer): "Century Gothic", "Avenir Next", "Trebuchet MS", sans-serif
- Body size: 11pt
- Line height: 1.65
- Heading weights: 700 (bold) for h1/h2, 600 (semibold italic) for h3
- Letter spacing: -0.2pt on h1, 3pt on eyebrow labels

### MLA Paragraph Formatting

- First-line indent: 24pt on all body paragraphs
- Text alignment: justified
- Serif font throughout body text
- No extra spacing between consecutive paragraphs (tight, flowing text)

### Creative Section Headers

Do NOT use generic headers. Use distinguished, gentlemanly alternatives:

| Generic Header | Creative Alternative |
|---|---|
| Executive Summary | A Note to the Contracting Office |
| Technical Approach | Our Approach and Methodology |
| Management Plan | How We Lead and Deliver |
| Staffing and Key Personnel | The People Behind the Work |
| Past Performance | A Record of Results |
| Quality Control | Our Standard of Excellence |
| Pricing Narrative | Investment and Value |
| Compliance Matrix | Compliance Traceability |
| Proposal | Scope of Services |

Headers should feel personal, confident, and professional - not corporate boilerplate.

### Layout

- Page: US Letter (8.5 x 11in)
- Margins: 22mm top, 25mm sides, 25mm bottom
- Cover page: dark cinematic (#111111) full-bleed background, centered logo, white title, crimson accent bars
- Body pages: warm cream (#FAF8F5) background, serif text, justified MLA paragraphs
- Chapter breaks: break-before: page on major sections
- Headings: 3pt crimson red left border on h1
- Tables: near-black header row with white text, alternating white/light-gray body rows

### Cover Page Structure

1. Logo: Thrare butterfly mark centered at top
2. Company name: "Rare Earth LTD" in Century Gothic, white
3. Subtitle: president name and DBA
4. Crimson accent bar separator
5. Eyebrow label: "GOVERNMENT PROPOSAL" (uppercase, lime green, 3pt letter-spacing)
6. Title: proposal title in serif, white, 28pt
7. Subtitle: "Prepared for [Agency] | Solicitation [Number]"
8. Certification tags: MBE, DBE, SDB, BAO - white text on lime green badges
9. Footer: address, phone, email, UEI, CAGE, EIN, due date
10. Bottom accent: crimson bar at page bottom

### Page 2: Letterhead + Introduction

1. Right-aligned letterhead: company name, certs, contact info
2. Crimson rule below letterhead
3. "A Note to the Contracting Office" (executive summary in MLA paragraph format)
4. Authorized Offeror Personnel block with lime green left border

### Section Structure (Proposal Body)

Body pages use cream background (#FAF8F5) with near-black text (#1A1A1A), serif font.

Follow this order for proposal sections:

1. A Note to the Contracting Office - opening letter to the buyer, personal and confident
2. Our Approach and Methodology - how we do the work, equipment, compliance
3. How We Lead and Deliver - PM governance, reporting, communication
4. The People Behind the Work - team, Eric White bio, staffing
5. A Record of Results - past performance with values, roles, outcomes
6. Our Standard of Excellence - QA/QC framework, corrective action
7. Investment and Value - cost methodology, rate basis
8. Compliance Traceability - requirement traceability table (appendix)

### What Goes in the Proposal PDF

- Executive summary as personal letter ("A Note to the Contracting Office")
- Authorized offeror personnel block
- All proposal body sections with creative headers
- Compliance traceability matrix (appendix)
- Company credentials footer

### What Does NOT Go in the Proposal PDF

- Bid readiness assessment (internal analysis only, stored in metadata)
- Compliance score percentage (internal metric)
- Internal notes, triage classifications, or lane assignments
- Morpheus scoring data

### Writing Rules

- Never use em dashes. Use hyphens (-) or commas instead.
- No placeholder brackets in final output
- All credentials must be real (UEI, CAGE, phone, email verified)
- Compliance matrix must include all identified requirements
- Past performance values must match verified contract records
- Cover page must include all four certification badges
- Footer must repeat: company name, DBA, certs, CAGE, UEI
- MLA paragraph format: indented first line, justified text, serif font
- Headers must use creative alternatives, never generic names
- Tone: confident, personal, gentlemanly - not corporate boilerplate

## AI Review Pipeline

Three tiers, all using Max subscription (no API key needed):

1. **Tier 1: Codex + Claude collaboration** - dispatch.sh with 10-min timeout, red-team review
2. **Tier 2: Claude CLI** - direct `claude -p` with context piped to stdin, 10-min timeout
3. **Tier 3: Deterministic fallback** - regex-based compliance scoring, section mapping

Tiers 1 and 2 use the Max subscription via Claude CLI. No Anthropic API credits required.

### Theme Verification (AI Review Step)

During AI review, the reviewer must verify the final HTML matches this brand spec:
- Cover page uses dark cinematic (#111111) background, not teal or parchment
- Headings use Bookman Old Style serif font, not sans-serif
- Accent bars and heading borders use crimson red (#CC1111)
- Cert badges use lime green (#6ABF1E) background
- Body text is near-black (#1A1A1A) on cream (#FAF8F5)
- Body paragraphs use MLA format (indented first line, justified)
- Section headers use creative alternatives (not "Executive Summary")
- No em dashes appear anywhere in the output
- All credential values match the Company Identity table above

If any deviation is found, the reviewer must flag it before PDF generation.

## Rendering Pipeline

1. Proposal Flow rough-draft stage generates markdown content
2. AI review stage improves quality and checks compliance (3-tier cascade)
3. Final-draft stage passes markdown + proposal metadata to `proposal-pdf.js`
4. `proposal-pdf.js` builds brand-styled HTML using the design tokens above
5. WeasyPrint converts HTML to print-ready PDF
6. HTML and PDF both saved to `server/data/exports/`
7. Server returns download URLs for both formats
