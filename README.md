# Proposal Flow

A modern web application for managing proposal development workflow.

## Features

- Proposal management
- Task tracking
- File organization
- Status tracking
- Deadline management

## Deployment

The application is deployed to GitHub Pages at: https://thrare00.github.io/proposal-flow/

## Development

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
